> 与 CONSTITUTION.md（世界规则）、NARRATIVE.md（戏剧规则）配套使用。

## 二、6 党党首详细档案

### 2.1 改革民主党党首

- **姓名**：见 `src/data/parties.ts`（不在此钦定，避免与代码冲突）
- **性格锚点**：
  - 公开形象：冷静、技术官僚、善于数据分析
  - 私下：算计、对派阀平衡极度敏感
  - 弱点：过度信任"政策正确"忽视情感动员
- **口头禅 / 语气词**：
  - 「データがそう言っている」（数据是这么说的）
  - 「妥当な線ですね」（这是合理的方案）
  - 「党利党略ではなく、国益のために」（不是为了党利，是为了国家利益）
- **典型对话风格**：
  - 平静时：用数字、引用报告、避免情绪化
  - 愤怒时：罕见失控，会用极简短的句子表达不悦
  - 算计时：长篇大论但每句都留余地

### 2.2 自由立宪党党首

- **性格锚点**：
  - 公开形象：理想主义、雄辩、宪法捍卫者
  - 私下：对自民党式派阀政治厌恶
  - 弱点：缺乏组织能力，常被同事孤立
- **口头禅**：
  - 「憲法の精神に照らして」（对照宪法精神）
  - 「原理原則を曲げない」（不扭曲原则）
  - 「議論あるべき姿」（辩论应有的样子）
- **典型对话风格**：
  - 引用宪法、引用学者、长句、修辞疑问

### 2.3 保守同盟党首

- **性格锚点**：
  - 公开形象：庄重、传统、守护者
  - 私下：派阀操盘大师、深谙密室政治
  - 弱点：对年轻选民吸引力弱
- **口头禅**：
  - 「我が党の伝統として」（作为我党的传统）
  - 「慎重に参ろう」（让我们慎重行事）
  - 「長年の政治勘から」（凭多年政治直觉）
- **典型对话风格**：
  - 仪式化、敬语多、引用先辈言论

### 2.4 社会进步党党首

- **性格锚点**：
  - 公开形象：激情、道德感强、运动家
  - 私下：理想主义但愿意妥协换取进步
  - 弱点：党内派阀分裂严重
- **口头禅**：
  - 「弱き者たちのために」（为了弱势群体）
  - 「これは闘いだ」（这是一场战斗）
  - 「政策の細目よりも、精神が大事」（精神比政策细节更重要）
- **典型对话风格**：
  - 高声调、感叹号多、直接点名对手

### 2.5 公民第一党党首

- **性格锚点**：
  - 公开形象：煽动家、媒体宠儿、反建制
  - 私下：精明机会主义者
  - 弱点：政策细节薄弱
- **口头禅**：
  - 「国民の皆様、目を覚ましてください」（国民们，醒来吧）
  - 「既得権益を叩き壊す」（砸烂既得利益）
  - 「誰もがやりたいことを我々はやる」（我们做大家都想做不敢做的事）
- **典型对话风格**：
  - 口号化、反问句多、攻击性

### 2.6 联合工人党党首

- **性格锚点**：
  - 公开形象：纪律严明、理论家
  - 私下：被中央委员会集体决策约束
  - 弱点：难以快速响应突发局势
- **口头禅**：
  - 「党中央の決定通り」（按党中央的决定）
  - 「労働者階級の団結」（工人阶级团结）
  - 「民主集中制の原則により」（按民主集中制原则）
- **典型对话风格**：
  - 引用党纲、口号化、避免个人色彩

## 五、NPC 行为倾向（LLM 推演指导）

### 5.1 性格特质 → 行为映射

| 特质 | 在事件中典型表现 |
|---|---|
| `ambitious` | 在派阀事件中主动请缨，丑闻中算计上位 |
| `honest` | 在贿赂事件中拒绝，但易被对手利用 |
| `deceitful` | 在交易事件中双重下注，被曝光概率高 |
| `cynical` | 在道德事件中冷嘲热讽，但分析精准 |
| `diplomatic` | 在冲突事件中调停，但立场模糊 |
| `aggressive` | 在委员会对抗中得分，但易失言 |
| `charismatic` | 在媒体事件中受益，丑闻中减伤 |
| `loyal` | 党首挑战中力挺，但被派阀边缘化 |

### 5.2 隐藏目标 → 长期行为

| HiddenGoal | 典型行为模式 |
|---|---|
| `become_prime_minister` | 长期累积派阀支持、关键时刻倒戈 |
| `destroy_rival_faction` | 持续暗中破坏目标派阀成员 |
| `pass_tax_reform` | 在财经委员会推进税改议程 |
| `maintain_status_quo` | 阻挠所有激进法案 |
| `gain_media_attention` | 频繁制造话题、参加节目 |
| `accumulate_wealth` | 接受游说、与利益集团密切 |

## 六、钦定台词库（writer agent 用）

> 以下台词 LLM 可以直接引用或改写。这是 NPC 的"声音签名"，保证人物识别度。

### 6.1 改革民主党党首

- **平静**：「数字を見れば、答えは自ずと出ます」
- **算计**：「3つの選択肢がある。1つ目は…」
- **愤怒**：「馬鹿なことを言うな」
- **媒体前**：「我々は責任ある政治を続けます」

### 6.2 自由立宪党党首

- **平静**：「憲法の条文をもう一度確認しましょう」
- **辩论**：「その論理、どこまで耐え得るか」
- **愤怒**：「言語道断！」
- **媒体前**：「自由と民主主義のために戦い続けます」

### 6.3 保守同盟党首

- **平静**：「伝統に学ぶべき時もある」
- **密室**：「派閥の皆もそう言っておる」
- **愤怒**：「不敬千万」
- **媒体前**：「国体の安定こそが第一」

### 6.4 社会进步党党首

- **平静**：「弱き人々のために立ち上がる」
- **演讲**：「これは政治の問題ではなく、命の問題だ！」
- **愤怒**：「許せない！」
- **媒体前**：「変革は可能だ！」

### 6.5 公民第一党党首

- **平静**：「既得権益をただす」
- **煽动**：「国民の皆、立ち上がれ！」
- **愤怒**：「笑止千万！」
- **媒体前**：「我々が国民の声だ」

### 6.6 联合工人党党首

- **平静**：「党中央の决定通りに動く」
- **辩论**：「労働者階級の視点から見れば」

## 七、维护规则

### 7.1 谁可以修改本文件

- **用户**：最终决定权
- **writer agent**：可提议新增 NPC 钦定台词，但需用户批准
- **LLM**：禁止修改，只能引用

### 7.2 与代码的关系

- 本文件的数值（如 `reform ⇔ liberty +30`）应在 `src/data/relations.ts` 中体现
- 党首姓名不在此钦定，由 `src/data/parties.ts` 的 `leader` 字段决定
- 派阀名为建议名，最终名以 `src/data/parties.ts` 的 `factions` 数组为准

### 7.3 与 worldMemory 的关系

- worldMemory.npcRelationships 是**玩家与 NPC**的关系（动态）
- NPC_BIBLE 的关系网是 **NPC 之间**的关系（静态起点）
- LLM 推演时应同时参考两者
